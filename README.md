## Rodando o programa

Para rodar o Projeto em um ambiente dockerizado, utilize o comando:

```bash
docker-compose up -d 
```
Após a execução do comando, o projeto será executado em um ambiente dockerizado.
Para Acessar entre no endereço: http://localhost:8000
